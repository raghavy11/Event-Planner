# 🎉 FunPlanner_v2

A modern event planning dashboard built with React, Tailwind CSS, and Framer Motion. Easily manage conferences, weddings, festivals, and more with a slick UI and popup-powered details.

---


## 🌐 Live Demo

[Live Link](https://event-planner-6dwm.vercel.app/)

---

## 🚀 Features

- **Interactive Hero Section**  
  - Animated SVG grid background responding to mouse movement  
  - Eye-catching headline + animated gradient text

- **Dynamic Client Logos Carousel**  
  - Automatically rotates logos every 5 seconds with smooth fade-in/out animations  
  - Hover overlay reveals “Meet our customers” CTA

- **Event Showcase Cards**  
  - Dark-themed image cards with immersive overlay effects  
  - Plus button animation to launch popups for additional details

- **Responsive Layout**  
  - Full-screen, centered hero section  
  - Grid system for cards (1–3 columns based on viewport size)

- **Metrics & Stats**  
  - Display of live stats: Teams, Uptime, Response Time  
  - Subtle hover animations for visual feedback

---

## 📦 Tech Stack

- **React** (with `useState`, `useEffect` hooks)  
- **Tailwind CSS** for utility-first responsive design  
- **Framer Motion** (`motion`, `AnimatePresence`) for polished animations  
- **lucide-react** for iconography (e.g., plus buttons)  
- Custom **Popup** component for detail overlays  
- SVG and CSS-driven animated background

## ⏳ Getting Started

1. **Clone the repository**  

```
   git clone https://github.com/raghavy11/Event-Planner
   cd FunPlanner_v2

Install Dependencies

    npm install
 or
    yarn install

Run The DEployment

    npm run dev
 or
    yarn dev

```
